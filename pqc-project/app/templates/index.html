Post-Quantum Cryptography Demonstration

Welcome to our demonstration of Post-Quantum Cryptography (PQC). In a world where quantum computers threaten to break our current encryption standards, PQC offers a new generation of cryptographic algorithms that are resistant to quantum attacks.

This website is a showcase of PQC in action. We are running two versions of this site:

  • Server A (Standard): This server uses traditional cryptography, which is vulnerable to attacks from quantum computers.
  • Server B (PQC-hardened): This server uses PQC to protect data, both in transit and at rest.

Explore the site and see the difference for yourself. You can also try our secure messenger to see PQC in action.
